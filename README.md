# C_Quiz_Game
A simple console quiz game written in C to practice core language concepts such as structs, pointers, and functions.

## Features
- Multiple choice questions
- Input validation
- Score calculation

## Compile and Run
```bash
gcc quizGame.c -o quizGame
./quiz game
```
## Concepts Practiced
- Structs
- Pointers
- Functions
- Arrays
